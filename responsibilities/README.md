# Responsibilities Folder

This folder is reserved for team responsibilities and evidence documentation.

## Recommended File Structure

- YYYYMMDD-memberId.md
  Member-based daily or sprint contribution notes
- evidence/
  Evidence files such as screenshots, logs, and reports

## Usage Rules

1. Each member should add dated entries to their own file.
2. Technical contributions should be clear and measurable.
3. Include build/test/endpoint evidence with links or output when possible.
4. Prefer concise, verifiable bullet points over long paragraphs.
